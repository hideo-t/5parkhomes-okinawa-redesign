# SEO監査メモ（2026-03-12）

## 1. 現状調査（旧サイト）
- **メタ情報不足**: `<title>` / `<meta name="description">` が未設定。検索結果で「Attention Required」等の Cloudflare 文言が表示される恐れ。
- **見出し構造なし**: ほぼすべての本文が太字段落で、`<h1>`〜`<h3>` の階層が存在しない。クローラがセクションを解釈しづらい。
- **キーワード密度の偏り**: 「トレーラーハウス」「沖縄」「オフグリッド」が高頻度で出現する一方、シナリオ別キーワード（ランドリー、BCP 等）は散発的。
- **画像の alt テキスト**: ほとんどが `Image 1` など汎用ラベル。内容を伝えられていない。
- **内部導線**: CTA が電話番号羅列のみで、構造化データやナビゲーションが不足。

## 2. リニューアルで実施した改善
- 50〜60文字のタイトルと 120〜155文字のディスクリプションを設定し、主要キーワード（トレーラーハウス／沖縄／BCP）を先頭に配置。
- `h1` を「沖縄トレーラーハウス専門店」で開始し、各セクションを `h2` / `h3` で整理。
- 画像に説明的な `alt` を付与。Markdown 由来の `Image X` も「Park Homes Okinawa トレーラーハウス写真」に差し替え。
- OG タグ／canonical を追加し、SNS や重複 URL 対策を実施。
- 旧サイト全テキストを `reference-site.md` 経由で表示しつつ、要約カード・FAQ でキーワードを自然に再配置。

## 3. 推奨キーワード
- メイン: `トレーラーハウス`, `沖縄`, `Park Homes`, `オフグリッド`, `BCP`, `ランドリー`
- サブ: `Solar Haven`, `Park Round`, `台風対策`, `助成金`, `離島`, `災害シェルター`

今後は `schema.org` の `LocalBusiness` / `Product` 構造化データや、ブログ型の長文記事を追加するとさらなる SEO 強化に繋がります。
